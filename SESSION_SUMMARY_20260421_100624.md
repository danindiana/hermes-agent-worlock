## Quick Start Guide

### Activate Virtual Environment
```bash
cd ~/hermes-agent
source venv/bin/activate
```

### Start Interactive Chat
```bash
hermes
```

### Change Model
```bash
hermes model
```

### View Current Configuration
```bash
hermes config show
```

### Edit Configuration
```bash
hermes config edit
```

## Key CLI Commands Reference

| Command | Purpose |
|---------|---------|
| `hermes` | Start interactive chat |
| `hermes model` | Select default model/provider |
| `hermes config show` | Display current settings |
| `hermes config set <key> <value>` | Update configuration |
| `hermes tools` | Configure tools per platform |
| `hermes setup` | Run full setup wizard |
| `hermes gateway` | Set up messaging (Telegram, Discord, etc.) |
| `hermes doctor` | Diagnose configuration issues |
| `hermes skills` | Manage skills |
| `hermes memory` | Configure external memory |
| `hermes mcp` | Manage MCP servers |
| `hermes sessions` | Manage conversation history |
| `hermes update` | Update to latest version |
